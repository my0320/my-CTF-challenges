# The Numbers

> The numbers... what do they mean?

## About The Challenge

We were given a picture which contain several numbers in .png file.

![preview](Images/the_numbers.png)

## How To Solve

1. I get the tools to get the numbers inside the picture and copy the numbers to identify its cipher.

(Recommended tools:[dcode](https://www.dcode.fr/cipher-identifier))

![preview](Images/preview1.png)

2. After knowing it is Letter Number Code, I used A1Z26 cipher to decode the numbers and get the flag.

(Recommended tools:[dcode](https://www.dcode.fr/letter-number-cipher))

![preview](Images/preview2.png)

3. The flag must be in the format 'picoCTF{}'.

```
PICOCTF{THENUMBERSMASON}
```
